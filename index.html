This personal project is designed to help me sleep, as I often struggle with insomnia.

The goal of this project is to document the various methods I have used to cope with insomnia over the years, even though I no longer suffer from it as often.

My primary solution was listening to horror stories, documentaries, and podcasts. This is a collection of all the content I have gathered, you can use it to help with the same issue or simply for fun.
